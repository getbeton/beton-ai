### Key Concepts for Consultants

* **Local vs. Remote**: You work on your "Local" copy. No one sees it until you "Push" it to the "Remote" (Cloud).
* **Commit**: This is just a "Save Point". It requires a short message (e.g., "Updated Q3 projections") so we have a history of *why* changes were made.
* **Push**: This "Publishes" your saved changes to the rest of the team.
* **Pull**: This "Refreshes" your library to get the latest work from your colleagues. **Always Pull before you start working!**
