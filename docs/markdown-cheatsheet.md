Markdown is a simple way to format text. Here are the most common tricks:

## Headers

Use `#` for titles.

```markdown
# Main Title (H1)
## Section Title (H2)
### Subsection (H3)
```

## Emphasis

* **Bold**: `**text**` -> **text**
* *Italic*: `*text*` -> *text*

## Lists

**Bulleted List**: Use `-` or `*`

```markdown
* Item 1
* Item 2
```

**Numbered List**: Use `1.`

```markdown
1. First step
2. Second step
```

## Links & Images

* **Link**: `[Click Here](http://google.com)`
* **Image**: `![Image Description](image-file.png)`

## Quotes
>
> This is a quote block.
> Just add a `>` at the start of the line.
